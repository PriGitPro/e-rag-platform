# Security Policy

## Reporting a vulnerability

Please report suspected vulnerabilities privately to:

- security@e-rag-platform.dev

Do not open public issues for security-sensitive reports.

## Response targets

- Acknowledgement: within 72 hours
- Initial triage: within 7 calendar days
- Remediation timeline: based on severity and exploitability

## Supported versions

At this stage, only the latest `main` branch is considered supported for security patches.
